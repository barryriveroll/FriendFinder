# **Friend Finder**

## Week 13 Homework Assignment, Coding Boot Camp, UNC-Chapel Hill

Originally submitted on 1/31/2019

### Instructions

- Answer 10 questions on a scale from Disagree to Agree.
- Your results are compared against others (and mock data) to show your best friend match.

### About

This project's emphasis is on express and uses a server to route to HTML pages and get data from an API via GET method and add data via POST method.

The data is also verified twice: once on the front-end, even though the basic inputs are fairly restricted
